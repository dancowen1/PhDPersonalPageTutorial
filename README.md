# Personal Academic Website — Workshop Template

A starter template for building your own personal academic website. Built for a workshop for humanities PhD students — no coding experience needed.

**Live demo:** https://phdpersonalpagetutorial.netlify.app/

## Start here

👉 **Read the [Workshop Guide](https://phdpersonalpagetutorial.netlify.app/guide/)** — it walks you through everything, including what to install and set up *before* the workshop.

Once you're ready to customise your own copy, [`CUSTOMIZE_ME.md`](CUSTOMIZE_ME.md) is your cheat sheet: it lists exactly which files to edit and what to change.

## Built with

- [Hugo](https://gohugo.io) — the static site generator
- [Blowfish](https://blowfish.page) — the theme (included as a git submodule)
- [Netlify](https://netlify.com) — hosting (push to GitHub → site updates automatically)

## Run it locally

```
hugo server
```

Then open http://localhost:1313 — the site rebuilds every time you save a file.
